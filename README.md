# HeyGroups

• Aplicativo de Chat em tempo real.

## Rodando localmente

Instale as dependências

```bash
  yarn
```

Abra o emulador e rode o projeto

```bash
  npx expo start
```


## Stack/Libs utilizadas

• Expo React Native

• Firebase

• Modalize
